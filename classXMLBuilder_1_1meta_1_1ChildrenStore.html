Detailed Description

template<class ParentType>
class XMLBuilder::meta::ChildrenStore< ParentType >

Store of children nodes.

Class for storage of children and implement interface for managing them

Childs are stored as vector of shared pointers of ChildableNode for keeping reference to the data of original type

Template Parameters
ParentTypeParent type of the actual node
